# 🚀 JCOVI Portfolio

> A futuristic, cyberpunk-inspired personal portfolio documenting my journey through software development, cybersecurity, networking, and cloud computing.

🌐 **Live Website:** https://jcovi.com

---

## Overview

This project serves as both my personal portfolio and a long-term development project. Rather than being a simple résumé website, it is designed to evolve alongside my career while demonstrating modern front-end development skills and interactive UI design.

Current focus areas include:

- 💻 Full-Stack Development
- 🔐 Cybersecurity
- 🌐 Networking
- ☁️ Cloud Computing

---

## Current Features

- Futuristic cyberpunk-inspired interface
- Responsive desktop and mobile layouts
- Animated particle background
- Glassmorphism UI
- Interactive sidebar navigation
- Animated terminal simulation
- Intro video modal
- Live system dashboard
- GitHub Pages deployment
- Custom domain (jcovi.com)
